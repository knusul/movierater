== movierater

Movierater is a clone of imbd
Written in rails4, backbonejs

== Requirements
* Ruby 2.1.3
* Rails 4.2
* Postgresql

== Setup
* copy config/database.yml.dist to config/database.yml
* run: bundle
* run: rake db:create db:migrate db:seet
* run: rails s
* enter in a browser: localhost:3000
